Python Sales Data Analysis Project
📌 Project Overview

This project demonstrates how to clean, analyze, and visualize sales data using Python.
It uses the powerful libraries pandas for data analysis and matplotlib for visualization.

The goal is to extract useful business insights such as:

Total revenue

Top-selling products

Category performance

Monthly revenue trends

🛠 Tools & Technologies

Python

pandas

matplotlib

CSV dataset

📥 Dataset

The dataset contains sales information including:

Column	Description
OrderID	Unique order number
Date	Date of purchase
Product	Product name
Category	Product category
Price	Product price
Quantity	Quantity sold
